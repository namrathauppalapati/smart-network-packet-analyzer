# Smart Network Packet Analyzer

A simple packet analyzer built using Python, Flask, and Scapy.

Features
- Capture live network packets
- Detect TCP, UDP, and ICMP protocols
- Store packet data using SQLite
- Display packets in a web dashboard
- Protocol statistics using Chart.js

Technologies
Python
Flask
Scapy
SQLite
HTML CSS JavaScript

Run the project

pip install -r requirements.txt
python app.py

Open browser

http://127.0.0.1:5000
